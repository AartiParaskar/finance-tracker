# 👑 Royal Finance Tracker

A modern and premium Personal Finance Tracker built using React, Tailwind CSS, Firebase, Framer Motion, and Recharts.

This project helps users manage daily expenses with beautiful UI, analytics charts, dark mode, authentication, and responsive design.

---

# 🚀 Features

## ✅ Authentication
- Firebase Login & Signup
- Secure User Authentication
- Logout System

## ✅ Expense Management
- Add Expenses
- Delete Expenses
- Category-wise Tracking
- Date-wise Expense Entry

## ✅ Dashboard
- Total Expenses
- Remaining Balance
- Budget Tracking
- Expense Analytics

## ✅ Charts & Analytics
- Pie Chart Analytics
- Category-based Visualization
- Responsive Charts

## ✅ Premium UI
- Royal Glassmorphism Design
- Dark / Light Mode
- Gradient Backgrounds
- Smooth Animations
- Responsive Layout

## ✅ Mobile Responsive
- Works on:
  - Mobile
  - Tablet
  - Laptop
  - Desktop
